 Shaheen Watch

A simple and elegant web-based watch/clock project built using HTML, CSS, and JavaScript.

 Features

- Real-time clock display
- Smooth animation
- Clean and minimal UI
- Lightweight and fast
- Works on all modern browsers

 Technologies Used

- HTML5
- CSS3
- JavaScript
